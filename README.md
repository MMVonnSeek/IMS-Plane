**IMS Plane** é um jogo educativo desenvolvido em Python com Pygame, inspirado na mecânica clássica de Flappy Bird. O projeto foi criado para ensinar programação a crianças de 9 anos, apresentando de forma visual e acessível conceitos fundamentais da lógica computacional.

---

## Objetivo do Projeto

O jogo foi concebido como ferramenta didática em sala de aula, permitindo que os alunos aprendessem Python enquanto interagiam com elementos reais de um game: movimento, obstáculos, pontuação e colisões. A experiência transforma teoria em prática, tornando o processo de aprendizado envolvente e significativo.

---

## Conceitos Técnicos Ensinados e Aplicados

- **Game loop contínuo**, estruturando a lógica de atualização e renderização  
- **Manipulação de sprites e imagens** com controles de posição  
- **Simulação simples de física**, utilizando velocidade, gravidade e momentum  
- **Detecção de colisões** para definir falhas e reinícios  
- **Geração procedural de obstáculos**, garantindo imprevisibilidade  
- **Eventos e entrada do usuário** com teclado (tecla espaço)  
- **Renderização de fontes e HUD**, exibindo pontuação em tempo real  
- **Estados de jogo**, organizando início, execução e fim  

Esses elementos oferecem uma base real de desenvolvimento de jogos, adequada para ensinar conceitos de programação de maneira concreta.

---

## Funcionamento do Jogo

O jogador controla um pequeno avião que sobe ao pressionar a barra de espaço. Tubos são gerados em posições aleatórias e deslocam-se pela tela, desafiando o aluno a manter o avião em voo e evitar colisões. A cada obstáculo superado, a pontuação aumenta, incentivando a persistência e o raciocínio rápido.

---

## Destaques Pedagógicos

- Tornar a lógica de programação **visual e compreensível**  
- Estimular **coordenação, lógica e tomada de decisão**  
- Demonstrar loops, condições e variáveis de forma prática  
- Motivar crianças a continuarem explorando tecnologia  

---

## Recursos Utilizados

O projeto utiliza imagens personalizadas, dois cenários diferentes, fontes customizadas, ícone da janela e variações gráficas para estados de vitória e derrota. Tudo isso contribui para um ambiente imersivo e atraente para o público infantil.

---

## Execução

Instale o Pygame:

```bash
pip install pygame
```

Execute o jogo:
python ims_plane.py
